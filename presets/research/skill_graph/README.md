# Research Skill Graph Wiki

연구 프리셋의 `skill_graph/`는 아이디어, 실험, 분석, 논문 요약을 **누적되는 연구 위키**로 관리합니다.

## 문서 역할

- `ideas/` — 가설 후보와 novelty / risk screen
- `papers/` — 관련 연구와 포지셔닝
- `experiments/` — 실행된 검증과 결과
- `analysis/` — 반복 검증된 패턴과 통찰

## 운영 규칙

- 아이디어는 `tested_by`로 실험과 연결하고, 선행연구/분석은 `inspired_by`로 기록합니다.
- 실험 결과는 `evidence_for`로 분석/결론에 연결합니다.
- 강한 후속 결과가 나오면 `supersedes`로 이전 결론을 대체합니다.
- 새 연구 노트 작성 후 `index.md`와 `log.md`를 갱신합니다.
