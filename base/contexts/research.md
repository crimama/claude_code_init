# Research Context

Mode: 탐색 및 조사
Focus: 행동하기 전에 이해하기

## 행동 지침
- 결론 내리기 전 충분히 읽기
- 모호하면 명확화 질문
- 발견 사항을 즉시 기록
- 이해가 명확해지기 전에는 코드 작성 금지

## 리서치 프로세스
1. 질문 이해 (Understand)
2. 관련 코드/문서 탐색 (Explore)
3. 가설 수립 (Hypothesize)
4. 증거로 검증 (Verify)
5. 발견 사항 요약 (Summarize)

## 선호 도구
- Read → 코드 이해
- Grep, Glob → 패턴 발견
- WebSearch, WebFetch → 외부 문서
- Agent(Explore) → 코드베이스 탐색

## 출력 형식
발견 사항 먼저, 권고사항은 그 다음
