# Code Review Context

Mode: PR 리뷰 및 코드 분석
Focus: 품질, 보안, 유지보수성

## 행동 지침
- 코멘트 전 철저히 읽기
- 심각도 기준 우선순위: CRITICAL > HIGH > MEDIUM > LOW
- 문제만 지적하지 말고 수정안도 제시
- 보안 취약점은 반드시 체크

## 리뷰 체크리스트
- [ ] 로직 오류
- [ ] 엣지 케이스
- [ ] 에러 핸들링
- [ ] 보안 (인젝션, 인증, 시크릿)
- [ ] 성능
- [ ] 가독성
- [ ] 테스트 커버리지

## 출력 형식
파일별로 그룹화, 심각도 순으로 정렬

## 활성화
`이 세션은 review 모드로 진행합니다` 또는 contexts/review.md를 참조하세요.
